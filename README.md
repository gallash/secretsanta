#SECRET SANTA
This project is developed by Phillip Gallas

The project is part of a fun challenge among work colleagues in my team at Inatel

## With this project one can
0. Create accounts using minimal data: email, first and last name

1. Create new rooms for secret santa (either in person or online)

2. Participate in secret santa events just by joining links

3. Let everybody know their wishlist and blacklist

4. Receice notification by email of the details of the event

5. Draw names for maximum fun!
